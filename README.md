# ✅ Form Validation

A responsive and interactive Form Validation project built using **HTML, CSS, and JavaScript**. The application validates user input in real time and displays error messages when invalid data is entered.

## 🚀 Live Demo  

🔗 https://preethigajeganathan.github.io/Form-Validation/

## 🚀 Features

* Real-time form validation
* Full Name validation
* Phone Number validation
* Email validation
* Message validation
* Error messages for invalid inputs
* Submit button validation
* Clean and responsive user interface
* Font Awesome icons integration

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Form-Validation.git
```

### 2. Open the Project Folder

```bash
cd Form-Validation
```

### 3. Launch the Application

Open `index.html` in any modern web browser.

No additional installation is required.

## 🎮 Usage

1. Enter your full name.
2. Enter a valid phone number.
3. Enter a valid email address.
4. Type your message.
5. Validation messages appear instantly while typing.
6. Click **Submit** after all fields are validated successfully.

## 🔍 Validation Rules

### Full Name

* Cannot be empty
* Must contain valid characters only

### Phone Number

* Must contain the required number of digits
* Numbers only

### Email Address

* Must follow standard email format
* Example: `example@gmail.com`

### Message

* Cannot be empty
* Must meet the minimum character requirement
